# Play Gear E-Commerce Site 

Play Gear is a MERN-based e-commerce platform focused on **sports, gym, fitness, and gaming products**.
---

## Tech Stack

### Frontend

* React (Vite)
* JavaScript
* CSS 

### Backend 

* Node.js
* Express.js
* MongoDB

---

## Project Structure

```
PlayGear_Nepal/
├── client/        # React frontend
├── server/        # Node + Express backend 
├── README.md
└── .gitignore
```

---


## Getting Started (Frontend)

```bash
cd client
npm install
npm run dev
```
