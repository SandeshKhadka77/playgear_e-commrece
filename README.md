# Play Gear E-Commerce Site 

Play Gear is a MERN-based e-commerce platform focused on **sports, gym, fitness, and gaming products**.
---

## Tech Stack

### Frontend

* React (Vite)
* JavaScript
* CSS (Tailwind planned)

### Backend 

* Node.js
* Express.js
* MongoDB

---

## Project Structure

```
PlayGear_Nepal/
├── client/        # React frontend
├── server/        # Node + Express backend 
├── README.md
└── .gitignore
```

---

## Development Workflow

* `main` → stable code
* Feature branches → individual features (navbar, cart, products)
* Each feature is developed, reviewed, and merged separately

---

## Getting Started (Frontend)

```bash
cd client
npm install
npm run dev
```
